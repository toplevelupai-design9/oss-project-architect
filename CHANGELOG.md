## [0.1.0] - 2026-07-14

### Added
- Project README with overview, structure, and quick-start guide.
- MIT license.
- Contribution guide with Conventional Commits and pull request process.
- Contributor Covenant v2.1 code of conduct.
- Security policy with responsible-disclosure process.

- AI prompt library covering triage, review, changelog, release notes, documentation, and commit messages.
- GitHub issue templates, pull request template, CODEOWNERS, and Dependabot configuration.
- Continuous integration workflows for Markdown linting and link checking, plus a stale-issue workflow.
